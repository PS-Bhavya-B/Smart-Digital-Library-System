# Smart-Digital-Library-System

## Description
The Smart Digital Library System is a Python-based application developed using Object-Oriented Programming (OOP) concepts. It manages a collection of books and provides recommendations based on the user's preferred genre.

---

## Features
- Add books to the library
- Display all available books
- Recommend books based on user preference
- Supports both regular books and eBooks

---

## OOP Concepts Used
- Classes and Objects (Book, EBook, User, Library)
- Encapsulation (use of private variables)
- Inheritance (EBook class inherits from Book)
- Polymorphism (method overriding for display function)

---

## Debugging
An intentional bug was introduced in the recommendation function.

### Issue
The system was recommending all books instead of filtering by genre.

### Resolution
The condition was corrected to match the book genre with the user's preferred genre.
